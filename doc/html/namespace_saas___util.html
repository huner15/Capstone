Detailed Description

A vector represents an array of type T for a fixed size N. While the underlying array operations and mechanics use std::array, the Vector class acts as a wrapper to allow for more readable vector field access using the dimensions x, y, z, and w appropriately for 2D, 3D, and 4D floating point vectors. Note that the public array member may be accessed externally to access underlying std::array functions. Vector is an aggregate class. A 3D vector may be initialized using the syntax Vector3D vector = {x, y, z}, where x, y, and z are floating point values. If less than the full number of arguments is added when constructing a vector, the remaining floating point values will be initialized to zero. Vector math operations may be added to Vector as needed.
