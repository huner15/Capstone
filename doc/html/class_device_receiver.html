Detailed Description

A DeviceReceiver is an abstract representation of an aircraft surveillance device data report receiver. Surveillance device data is expected to received on separate TCP/IP network ports and this abstract receiver is designed to exist in it's own thread. Receiving behavior is dependent on te behavior of ReceiveReport() for a specific device type.
